## Project Description

A comprehensive help desk solution for managing support tickets with role-based access (Customer, Agent, Admin), offline functionality, realtime notifications, and cross-platform compatibility.



## Architecture

Cross-platform help desk system with React Native mobile app and Next.js web app, sharing Express.js backend with PostgreSQL, Prisma ORM, Socket.io for realtime, and offline sync capabilities.
