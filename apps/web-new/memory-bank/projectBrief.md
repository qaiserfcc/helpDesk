## Project Summary

Deliver a cross-platform help desk solution including mobile app and web application that lets customers and internal staff log, assign, track, and resolve support tickets with offline capability, push/email notifications, and a secure Express/PostgreSQL backend. The web app provides desktop UI with functional parity to the mobile app.
