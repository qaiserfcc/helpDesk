## Permission-based Access Control

Role-based permission system using string-based permissions (e.g., 'tickets:create', 'admin:manage_users') with hasPermission and getRolePermissions utilities. Zustand stores for global state management with persistence. TanStack Query for server state with offline queuing. Socket.io for realtime ticket updates with auth refresh on errors.
